1️⃣ Penjualan

    Form input barang + keranjang (sudah ada).

    Edit qty di keranjang (kalau pembeli minta ubah jumlah tanpa hapus barang).

    Hapus item di keranjang (sudah kamu minta tadi).

    Diskon per item & diskon total.

    Pajak otomatis (kalau ada PPN).

    Simpan transaksi → masuk ke database penjualan.

    Cetak/unduh struk (PDF atau thermal printer).

    Riwayat penjualan → untuk melihat transaksi hari ini, kemarin, dll.

    Pencarian transaksi → berdasarkan nomor struk atau tanggal.

2️⃣ Stok

    Tambah stok (restock barang).

    Edit data barang (harga, nama, kategori).

    Peringatan stok minimum (notifikasi kalau stok mau habis).

    Riwayat mutasi stok (log perubahan stok).

    Import/export stok (Excel/CSV).

3️⃣ Pengeluaran

    Kategori pengeluaran (misal: listrik, sewa, gaji, transport).

    Lampiran bukti pengeluaran (opsional, upload gambar/nota).

    Tanggal pengeluaran (supaya bisa filter bulanan).

    Filter & rekap per periode.

4️⃣ Laporan

    Laporan penjualan harian/bulanan.

    Laporan pengeluaran harian/bulanan.

    Laporan laba rugi → total penjualan - total pengeluaran.

    Grafik performa (penjualan per hari, per produk, tren stok).

    Produk terlaris.

    Produk kurang laku.

5️⃣ Keamanan & Pengaturan

    Login user (kasir, admin).

    Hak akses → kasir hanya bisa input transaksi, admin bisa edit stok & lihat laporan.

    Backup database otomatis/manual.

    Pengaturan toko (nama, alamat, logo).
